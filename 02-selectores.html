01 - Selector de etiqueta (1pt)

El selector de etiqueta permite modificar la apariencia de las etiquetas indicadas en CSS. Por ejemplo a continuación se modifica una etiqueta de enlace para que luzca distinta.

Enlace 
        
/* Selector de etiqueta */
a{
    text-decoration: none;
    color: black;
    font-weight: bold;
    font-family: Arial, Helvetica, sans-serif;
    background-color: antiquewhite;
    padding: 10px;
    border-radius: 5px;
    display: block;
    width: min-content;
    margin: 0 auto;
    text-transform: uppercase;
}
body{
    font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
    margin: 10px 20px 0px 20px;
}
pre{
    margin: 15px;
    background-color: rgb(20, 20, 20);
    padding: 30px;
    border-radius: 15px;
}
code{
    color: springgreen;
}

02 - Selector de clase (10pts)

El selector de clase utiliza el atributo class="" para otorgar estilos sobre las clases indicadas en CSS. Por ejemplo el siguiente parrafo posee la clase "parrafo-principal".

Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolor, commodi! Perferendis fugit iusto quibusdam suscipit excepturi, quidem dolorem numquam nemo natus laboriosam ullam, tenetur iure! Expedita magnam laboriosam cumque unde.

        
/* Selector de clase */
.parrafo-principal{
    display: block;
    margin: 30px;
    padding: 20px;
    border-radius: 5px;
    background-color: rgb(3, 3, 53);
    color: white;
    text-align: center;
}

03 - Selector ID (100pts)

El selector de ID no es muy recomendado de utilizar debido a su alta especificidad, es usado principalmente para generar enlaces internos en el documento HTML. En el siguiente ejemplo otro parrafo mediante selector ID.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Asperiores, itaque recusandae. Cumque consequuntur excepturi, repellat aut, aspernatur molestias facilis temporibus praesentium accusamus adipisci debitis harum officia tenetur autem, incidunt sit!

        
/* Selector por ID */
#parrafo-secundario{
    text-align: justify;
    font-family: monospace;
    color: springgreen;
    background-color: black;
    padding: 30px;
    margin: 10px;
    border-radius: 20px;
}

04 - Selector Global

Permite modificar el estilo de todo el documento HTML, tambien es replicable utilizando un selector en body o en html. En el ejemplo se modifico el color de todos los textos por defecto a gris oscuro.

        
/* Selector global (*) */
*{
    color: rgb(39, 39, 39);
}

05 - Selector descendiente

Permite dar estilos a etiquetas según su anidamiento, es decir, aplica estilos a las etiquetas que compartan el orden descrito en CSS. Como ejemplo se modificara la siguiente lista.

  1. Pizza
  2. Sushi
  3. Hamburguesas
        
/* Selector descendiente */
ol{
    background-color: steelblue;
    border-radius: 5px;
    text-align: center;
}
ol li{
    list-style: none;
    display: inline-block;
    padding: 10px 20px 10px 10px;
    font-weight: bold;
}

06 - Selector Multiple

Selector que aplica estilos a todos los elementos indicados separados por coma. En el ejemplo le cambiaremos el estilo a dos elementos con distintas clases.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Quia, porro provident adipisci recusandae exercitationem error, non harum numquam ut tempore praesentium, temporibus officia labore iusto? Fuga officiis accusantium necessitatibus sit?

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Excepturi nemo voluptatum officia earum optio in quas placeat aspernatur pariatur adipisci aliquid recusandae, illo doloribus! Officiis deleniti atque maiores temporibus eius!

        
/* Selector Multiple */
.multi-a, .multi-b{
    background-color: tomato;
    padding: 15px;
}

07 - Selector hijo directo

Aplica estilos a todos los elementos siguientes del elemento indicado en CSS. En el ejemplo se le aplican estilos a los elementos hijos de una lista.

  • Primer elemento
  • Segundo elemento
    • Primer elemento interno
  • Tercer elemento
        
/* Selector hijo directo */
.hijo-directo > ul > li{
    background-color: rgb(255, 148, 148);
    font-weight: bold;
    color: black;
}

08 - Selector adyacente

Aplica estilos al primer elemento inmediatamente siguiente indicado en CSS. En el ejemplo modificamos el estilo del primer parrafo adyacente a una lista.

  • Primer elemento
  • Segundo elemento

Primer texto

Segundo texto

        
/* Selector Adyacente */
ul + p {
    background-color: black;
    padding: 10px;
    display: inline;
    color: white
    }

9 - Selector de atributos

Aplica estilos según el atributo que posea la etiqueta indicada. En el ejemplo destacamos los parrafos que posean el atributo class y el valor "alexis".

Maria

Alexis

Felix

        
/* Selector de atributo */
p[class="alexis"]{
    background: red;
    display: inline;
    padding: 15px;
    font-weight: bold;
}

10 - Selector X:visited

Estos selectores mediante pseudoelementos permiten modificar el estilo cuando un enlace a sido visitado.

VISITED 
        
/* Selector de pseudoelemento visited */
.pseudo-link{
    display: block;
    text-align: center;
}
.visited:visited{
    background-color: black;
    border: 2px solid white;
    color: white;
    border-radius: 15px;
    display: inline;
}
